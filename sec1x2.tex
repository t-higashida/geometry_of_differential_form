\section{多様体の定義と例}
\subsection{局所座標と位相多様体}

$M$を位相空間としよう．いわば一番広い意味での図形である．
これから，$M$が多様体になるための条件を順に追ってあげていくことにする．

まず$M$は\textbf{Hausdorffの分離公理}を満たすものとする．
すなわち，任意の異なる2点$p,q	\in	M$に対し，$p$の開近傍$U$と$q$の開近傍$V$とが
存在して，$U$と$V$とが互いに交わらないようにできるとする．
このような$M$を\textbf{Hausdorff空間}という，この分離公理を満たさない重要な
位相空間が無いわけではないが，
\footnote{
例えば代数幾何学で取り扱うザリスキー位相（Zariski topology）は大変重要．
}
幾何学の対象になる通常の図形は殆どこの公理を
満たしているといてよい．たとえば数空間$\R^n$やその部分空間はすべて明らかに
Hausdorff空間である．

第二の条件は，$M$の任意の点$p$に対し，$p$のある開近傍$U$が存在して，それが$\R^n$の
ある開集合$V$と同相になっているというものである．$\varphi	:	U	\rightarrow	V$
を同相写像としよう．このとき$U$上の各点$q$に対し，その$\varphi$による像
$\varphi(q)$は$\R^n$の点であるから
\[
	\varphi(q)	=	(x_1(q)	,	\cdots	,	x_n(q))
\]
と$n$個の実数の組で表すことができる．この組のことを点$q$の
\textbf{局所座標}（local coordinates），また$U$を\textbf{座標近傍}
（coordinate neighborhood）という．
また$x_1,	\cdots	,	x_n$のことを$U$上で定義された\textbf{座標関数}という．
このようにして，
\begin{figure}[hbtp]	
		\centering	
		\includegraphics[bb=0 0 800 600,clip,width=8cm,height=3.5cm]{fig1x7.bmp}	
		\caption{局所座標系（chart）}	
		\label{fig1x7}	
\end{figure}	
$p$の十分近くの点はすべて局所座標と呼ばれる$n$個の互いに独立なパラメーター
で一意的に表すことができることになる．組$(U,	\varphi)$のことを
\textbf{局所座標系}（local chart または local coodinate system）と呼ぶ．
$(U,	\varphi)$の代わりに座標関数を表に出して$(U;x_1,\cdots	,	x_n)$と書く場合
もある．ここでchartとは“地図”を意味する言葉である．$(U,	\varphi)$または
$(U;	x_1,	\cdots	,	x_n)$により点$p$の近くを表す地図が与えられたというのである．

つぎに，$M$の位相に関するもう一つの条件を仮定する．この条件は，
\textbf{第二可算公理}（second countability axiom）と呼ばれているもので，
可算個の元からなる開集合系の基が存在するというものである．
すなわち，$M$の可算個の開集合$U_1,U_2,	\cdots	$があって，
任意の開集合$U$とその上の点$p$に対し，ある$i$が存在して
$p	\in	U_i	\subset	U$となるようにできるというのである．
$\R^n$上の有理点（座標がすべて有理数であるような点）と，それらの点の
正の有理数$\varepsilon$に対する$\varepsilon$近傍をすべて考えることにより，
$\R^n$がこの公理を満たすことがわかる
\footnote{
有理点も有理数近傍も可算個
}．
多様体を定義するときには，通常この公理をはじめからは仮定しない場合が
多い．しかし，ふつう扱われる多様体は殆ど例外なくこの公理を満たしているので，
本書ではこの条件をはじめから仮定することにする．

以上の三つの条件を満たす位相空間を位相多様体という．

\定義{1.4}（位相多様体）
第二可算公理を満たすHausdorff空間$M$は，
その上の任意の点が$\R^n$の開集合と同相になるような開近傍をもつとき，
$n$次元\textbf{位相多様体}（topological manifold）であるという．
\□

\例{1.5}
上の三つの条件のうち，Hausdorffの分離公理だけをみたさない位相空間の簡単な例を
あげよう．平面$\R^2$の部分集合$M$を
\[
	M=\{\:	(x,0);	x	<	0	\:\}	\cup	
		\{\:	(x,1);	x	\geqq	0\:\}	\cup
		\{\:	(x,-1);	x	\geqq	0\:\}
\]
と定義する(図\ref{fig1x8})．$M$に平面の部分空間として位相とは異なる次のような
位相を入れる．
\begin{figure}[hbtp]	
		\centering	
		\includegraphics[bb=0 0 800 600,clip,width=6cm,height=3cm]{fig1x8.bmp}	
		\caption{}	
		\label{fig1x8}	
\end{figure}	

2点$p_+=(0,1),p_-=(0,-1)$以外の点$p	\in	M$については
通常の$\varepsilon $近傍$U_\varepsilon (p)	=	\{	q	\in	M	\;	;	\;	d(p,q)	<	\varepsilon \}$
を考えて，除外された2点については
\[
	U_\varepsilon (p_{\pm})
	=
	\{	(x,0)				\;	;	\;	-\varepsilon <	x	<	0	\}	\cup
	\{	(x,\pm	1)	\;	;	\;	0 <	x	\leqq	\varepsilon	\}
	\quad	\text{(複合同順)}
\]
とおく．このとき，$M$上のすべて点$p$とすべての正数$\varepsilon $に対する
$U_\varepsilon(p)$の全体の開集合の基とするような位相を入れるのである．
定義から2点$p_+,p_-$の開近傍をどのようにとっても，それらは必ず交わってしまう．
したがって$M$はHausdorff空間ではない．しかし，$M$が他の二つの条件を
満たすことは明らかであろう．
\□
